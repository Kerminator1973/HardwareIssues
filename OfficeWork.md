# Компьютеры в офисе

В августе 2024 года столкнулся с ситуацией, когда несколько сидящих рядом сотрудников использовали беспроводные мышки и клавиатуры. Из-за "эфирного шума" при работе с мышью начинали возникать различные глюки - систематическое "не срабатывание" клики, потеря выделения текста при перемещении курсоры мыши, и т.д.

Проблему, гипотетически, можно решить выбрав не конфликтующие диапазоны частот. На практике же, единственный рабочий вариант - проводные мыши и клавиатуры.

## Пропустить создание учётной записи Microsoft на этапе установки Windows

Пропустить создание учётной записи Microsoft на этапе установки можно. Достаточно нажать Shift+F10, ввести в командную строку `start ms-cxh:localonly` и в появившемся окошке создать локальную учётную запись.

## Удалить кривое обновление Microsoft Windows

Начиная с 2024 года, Microsoft стала публиковать значительное количество обновлений системы, которые нарушают работоспособность системы. Чтобы откатить кривой update, следует зайти в режим восстановления Windows и выполнить команду

```shell
wusa /uninstall /kb:5077181 /quiet /norestart
```

Параметр `/kb:` указывает на пакет обновлений, который необходимо удалить.
