# Мышка для ноутбука Peeble Mouse 2

Мышка продолжает очень удачную серию Peeble Mouse M350, которая отличается компактными размерами, верхнюю крышку на магнитах и поддержкой соединения по радио-каналу и Bluetooth.

Обновление мышки - Peeble Mouse 2 M350S отличается тем, что:

- содержит более точный оптический сенсор 4000 DPI vs 1000 DPI
- тихие свичи на кнопках мыши
- поддерживает новый тип ресивера - **Logi Bolt**
- выпускается из переработанного пластика (утверждается о 76% повторно используемого пластика)

В коробку с мышкой не включён ресивер, его нужно докупать отдельно. "Из коробки" мышка работает по Bluetooth.

Чтобы начать подключение по Bluetooth необходимо выбрать средний индикатор (2) и держать кнопку переключения нажатой в течение 3 секунд.
